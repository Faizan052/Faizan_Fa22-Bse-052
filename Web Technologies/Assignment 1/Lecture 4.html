Tables are used in HTML to display information in a structured manner using rows and columns.

They help organize data efficiently and are commonly used in pricing lists, schedules, reports, and databases.

A table consists of table rows (<tr>), table headers (<th>), and table data cells (<td>).

Example:

Product Price Stock
Smartphone $500 Available

The <div> element is a block-level container that is used to group elements together and apply CSS styles.

It helps structure the page layout, allowing better content management and improved responsiveness.

Example:

Product Details

This section contains product information inside a styled div.

ðŸ“Œ Span Element

The <span> element is an inline container used for styling specific portions of text.

Unlike <div>, which is block-level, <span> does not create a new line and is used within text elements.

Example:

The price is $500 and is on sale!
