HTML Forms are used to collect user input in web applications. They allow users to enter data, which is then sent to a server for processing.

Common use cases include login pages, registration forms, contact forms, and surveys.

ðŸ“Œ The <form> Element

The <form> element is the container for all form fields. It defines where and how the user data should be submitted.

Example:

                <form action="/submit" method="post">
                    
                </form>

ðŸ“Œ Form Elements & Attributes

  • action: Specifies the server URL where the form data is sent.
  • method: Defines the HTTP method (GET or POST).
  • input: Used for text fields, checkboxes, radio buttons, etc.
  • label: Connects text with an input field.

ðŸ“Œ Example: A Basic Form

                <form action="/submit" method="post">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name" required>
                    
                    <label for="email">Email:</label>
                    <input type="email" id="email" name="email" required>
                    
                    <button type="submit">Submit</button>
                </form>
